Given a time in AM/PM format, convert it to military ($24$-hour) time.

Note: Midnight is $\text{12:00:00AM}$ on a $12$-hour clock, and $\text{00:00:00}$ on a $24$-hour clock. Noon is $\text{12:00:00PM}$ on a $12$-hour clock, and $\text{12:00:00}$ on a $24$-hour clock.

Input Format

A single string containing a time in $12$-hour clock format (i.e.: $\text{hh:mm:ssAM}$ or $\text{hh:mm:ssPM}$), where $01 \le hh \le 12 $.

Output Format

Convert and print the given time in $24$-hour format, where $00 \le hh \le 23 $.

Sample Input

07:05:45PM

Sample Output

19:05:45
